# Employee Management System

## Project Description
This is a full stack web application to manage employee records. Users can login and perform add, update, delete, and search operations.

## Features
- Login and Logout
- Add Employee
- Edit Employee
- Delete Employee
- Search Employee

## Tech Stack
- Python Flask
- MySQL
- HTML, CSS
- Git & GitHub

## How to Run
1. Install required packages:
   pip install flask mysql-connector-python

2. Run the application:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000/login
